Purpose

This site details info about some basic data types in Python like integers and lists. It also wants to inform about the difference between "passing by value" and "passing by reference" as different data types behave differently, as when you're not expecting it, it can make your code behave in a weird manner.

Passing by value

When we pass a variable through a function, the function will make a new spot in memory and hold the value of the variable that was passed on there, effectively making a copy of the original variable, this means that any changes made to the variable in the function will not affect the original one.

Passing by reference

When we pass a variable through a function, the function will receive the location in memory of the variable, this we call a "reference". Because the function receives the same memory spot that the variable looks at, it means that any changes we make to that variable in the function will affect the original variable as well.
